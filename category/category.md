# Todo Category Feature

Author: Dinh Minh Ngoc Thang  
Role: Reviewer  

## Overview
The Todo Category feature allows users to organize their tasks by assigning
each todo item to a specific category. This helps users manage their work
more efficiently and keep related tasks grouped together.

## Description
With this feature, a todo can belong to one category at a time.
Categories help users distinguish between different types of tasks
such as work-related tasks, study plans, or personal activities.
This improves readability and makes the todo list easier to maintain.

## Main Functions
- Create categories for todos
- Assign a category to a todo
- Update the category of an existing todo
- Remove or change categories when needed

## Example Categories
- Work
- Study
- Personal
- Health
- Shopping

## Benefits
- Better organization of todos
- Faster task management
- Clear separation between different types of tasks
## Notes
- Each todo should belong to only one category at a time
- Categories can be extended in the future
- This feature improves clarity and usability of the todo list
