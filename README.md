## Tech Stack
- Frontend: Next.js
- Backend: Node.js
- Database: MongoDB

## Quy trình làm việc
- Branch `main`: Production code
- Branch `develop`: Development code
- Branch `feature/*`: Feature branches cho từng thành viên
- Branch `hotfix/*`: Khẩn cấp sửa lỗi

## Cài đặt
```bash
# Clone repository
git clone https://github.com/vuong20031591-hub/todolist.git

# Cài đặt dependencies
npm install

# Chạy ứng dụng
npm start
```
